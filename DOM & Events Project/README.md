# Caesar Cipher App 

This app implements a Caesar Cipher that shifts each character in a string by a user-defined value. 
It is a simple application created to practice JavaScript logic, Git workflow, and using GitHub for version control.

## Tech Stack 
* HTML 
* CSS
* JavaScript 

## Installation 

Clone the repository:

1. Git Bash → git clone <repo-url>
2. Open index.html in your browser. Alternatively use "Live Preview" extension on Visual Studio Code.
