# Semantic HTML Project

This project focuses on practicing the use of semantic HTML elements to create a well-structured webpage.
It was created as part of my front-end reviewing process to better understand how semantic tags improve readability, accessibility, and page structure.

## Tech Stack

* HTML
* CSS

## Installation

Clone the repository:

Git Bash → git clone <repo-url>

Open `index.html` in your browser.
Alternatively, use the **Live Preview** extension in Visual Studio Code.

## What I Reviewed

* How to structure webpages using semantic HTML elements
* The difference between semantic tags and generic containers like `<div>`
* How semantic markup improves accessibility and readability
* Organizing HTML and CSS for small front-end projects
