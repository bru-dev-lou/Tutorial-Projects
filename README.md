# Tutorial Projects

This repository contains small projects that I completed while following tutorials or guided learning resources.

The goal of this repository is to document my learning process and practice implementing new concepts in real code. While some guidance was used during these projects, all code was written and reviewed by me to reinforce the concepts being learned.

These projects focus on strengthening my understanding of front-end and full-stack development fundamentals.

- Build confidence with development tools and workflows
- Document my learning progress
