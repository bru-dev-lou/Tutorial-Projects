# Pixel Paws Project

This project focuses on building a small interactive virtual pet while practicing how JavaScript classes can be used to structure application logic. It was created as part of my JavaScript practice to better understand how classes organize behavior and manage object state in a small front-end project.

## Tech Stack

* HTML
* CSS
* JavaScript

## Installation

Clone the repository:

* Git Bash → git clone
* Open index.html in your browser. Alternatively, use the Live Preview extension in Visual Studio Code.

##  What I Practiced

* Creating and using JavaScript classes
* Managing object state with class properties and methods
* Handling user interactions with event handlers
* Connecting JavaScript logic with HTML elements
